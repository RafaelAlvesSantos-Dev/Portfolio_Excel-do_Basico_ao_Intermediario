O que foi este curso?

Capacitar na manipulação de grandes volumes de dados, automação de cálculos com funções avançadas e geração de relatórios dinâmicos para tomada de decisão.

Os Módulos de Excel

1 Tabelas de Dados

Diferenças entre intervalos e tabelas, formatação, linha de totais e gestão de nomes.

estruturação
// 1.1 a 1.3: Definições e Totais
- Intervalo: Dados soltos (A1:C10)
- Tabela: Objeto estruturado (Ctrl+Alt+T)
- Linha de Totais: Soma/Média automática

// 1.9: Nomeando o objeto
Design da Tabela > Nome da Tabela: "Tab_Vendas"
1.4 - 1.6 Manipulação de Dados

Técnicas de filtragem I e II, segmentação (Slicers) e classificação de A-Z.

filtros
// Segmentação de Dados
Inserir > Segmentação > [Coluna]
(Cria botões visuais para filtrar)

// Classificar
Dados > Classificar > Por Cor ou Valor
1.7 - 1.8 Conversão e Tamanho

Como alterar o tamanho da tabela e converter de volta para intervalo comum.

ajustes
// Converter em Intervalo
Design da Tabela > Converter em Intervalo
(Remove a inteligência, mantém a cor)

// Redimensionar
Arrastar o canto inferior direito da tabela.
2 Referências (Partes I, II e III)

Domínio total do travamento de células com o símbolo de cifrão.

fórmulas
// Parte I: Relativa
=B2 * C2 

// Parte II: Absoluta (Total)
=B2 / $G$10 (Trava linha e coluna)

// Parte III: Mista
=B2 * $A5 (Trava apenas a coluna A)
3 Vinculação de Planilhas

Como buscar e somar dados que estão em abas diferentes do arquivo.

vínculos
// Sintaxe de outra aba
='Nome da Aba'!Célula

// Exemplo Parte III
=SOMA('Janeiro:Dezembro'!B10)
4 PROCV e Variações

Uso com tabelas estruturadas e busca aproximada (Verdadeiro).

busca
// 4.1: Com nomes de tabela
=PROCV("ID_01"; Tab_Vendas; 2; 0)

// 4.2: PROCV VERDADEIRO (Aproximado)
=PROCV(Nota; Tab_Conceitos; 2; 1)
5 Lógica Condicional

Tomada de decisão automatizada baseada em critérios.

lógica
// Sintaxe
=SE(Teste; Valor_Se_Sim; Valor_Se_Não)

// Exemplo
=SE(Estoque < 10; "Repor"; "OK")

A Jornada em Resumo

Módulo 1 · Início
Estruturação e Organização
Aprendi que o Excel é muito mais que células. Transformei dados "soltos" em Tabelas inteligentes e aprendi a classificar e filtrar informações com agilidade.
Módulo 1 · Fundamentos
O Poder das Referências
Entendi a diferença crucial entre referências relativas e absolutas ($). Isso me deu a liberdade de criar fórmulas que podem ser arrastadas sem quebrar os cálculos.
Módulo 2 · Lógica
Busca e Tomada de Decisão
Com o PROCV e a Função SE, aprendi a fazer o Excel trabalhar por mim, automatizando buscas entre planilhas e criando alertas inteligentes baseados em critérios.
Módulo 2 · Análise
Inteligência de Dados
Finalizei dominando Tabelas Dinâmicas. Consegui resumir milhares de linhas em apenas alguns cliques, gerando visões claras para análise de resultados.
